# MindfulSEO WordPress Plugin

**Version:** 1.0.0  
**Author:** [Mindful Design](https://mindfuldesign.me)  
**Website:** [mindfuldesign.me](https://mindfuldesign.me)

---

AI-powered SEO optimization plugin for WordPress. Batch optimize titles, descriptions, and keywords with OpenAI or Claude while respecting your brand guidelines.

## Features

### Core Functionality
- **AI-Powered Optimization**: Uses OpenAI GPT-4 or Claude to generate SEO titles, meta descriptions, and keywords
- **Batch Processing**: Optimize hundreds of posts at once with intelligent queuing
- **SEO Audit Dashboard**: Instantly identify posts with SEO issues (missing metas, keywords, low scores)
- **Keyword Strategy**: Build and manage targeted keyword lists with search volume data
- **Language Guidelines**: Define brand voice rules that AI must follow
- **DataForSEO Integration**: Get real search volume, keyword difficulty, and CPC data
- **Works with Rank Math & Yoast**: Seamlessly integrates with popular SEO plugins

### Key Features
- **Smart Keyword Extraction**: 4-strategy system identifies the best keywords from your content
- **Title Preservation**: AI keeps your core topic while optimizing for SEO
- **One-Click Fix**: SEO Audit → Fix All → Batch Optimizer workflow
- **Inline Editing**: Edit any field directly in the table
- **30-Day Caching**: DataForSEO data caches to save API costs
- **Validation System**: Automatic checks for title/description length

### Perfect For
- Content-heavy websites
- Bloggers and publishers
- Marketing agencies
- E-commerce sites
- Educational platforms
- Any WordPress site needing SEO optimization at scale

## Installation

1. Upload the `mindfulseo` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **MindfulSEO → Settings** to configure your API keys
4. Add your OpenAI or Claude API key and test the connection
5. (Optional) Add DataForSEO credentials for keyword metrics

## Requirements

- WordPress 5.8 or higher
- PHP 7.4 or higher
- Rank Math SEO or Yoast SEO plugin (recommended)
- OpenAI API key OR Claude API key
- DataForSEO account (optional, for keyword metrics)

## Quick Start

1. **Settings**: Add your AI API key (OpenAI or Claude)
2. **SEO Audit**: See all your SEO issues at a glance
3. **Click "Fix All"**: Posts auto-select in Batch Optimizer
4. **Optimize**: AI generates SEO-optimized titles, descriptions, and keywords

## Configuration

### API Keys
- **OpenAI**: Get your key from [platform.openai.com](https://platform.openai.com)
- **Claude**: Get your key from [console.anthropic.com](https://console.anthropic.com)
- **DataForSEO**: Sign up at [dataforseo.com](https://dataforseo.com) (optional)

### Language Guidelines
Import your brand guidelines or let AI generate them from your existing content. Rules include:
- Terms to avoid (e.g., "cheap" → "affordable")
- Proper capitalizations
- SEO-friendly phrasings

### Keyword Strategy
- Import keywords from CSV
- Auto-generate with AI
- Get DataForSEO metrics (volume, difficulty, CPC)
- Manual keyword entry
